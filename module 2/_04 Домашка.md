* "сколько бензина в баке" (gas)
* "вместимость бака" - сколько максимум вмещается бензина (capacity)
* "расход топлива на км" (gas_per_km)

``` python
car1.fill(5)  # залили 5 литров
```

``` python
car1.ride(50)  # едем 50 км (если хватит топлива)
```
