```python
class Point:
   def __init__(self, x, y):
       self.__x = x
       self.__y = y
```

### способ № 1 Примитивный

```python
def get_coords(self):
     return self.__x, self.__y

def set_coords(self, x, y):
     self.__x = x
     self.__y = y


A = Point(1, 2)
print(A.get_coords())

A.set_coords(3, 4)
print(A.get_coords())
```

### способ № 2 Через функцию property

```python
def get_coords(self):
  return self.__x, self.__y

def set_coords(self, x, y):
    self.__x = x
    self.__y = y


coords = property(get_coords, set_coords)

A = Point(1, 2)

print(A.coords)

```

### способ № 3 Через декоратор @property

```python
@property
 def x(self):
    return self.__x
@x.setter
def x(self, x):
       self.__x = x

A = Point(1, 2)
A.x = 5
print(A.x)

```
