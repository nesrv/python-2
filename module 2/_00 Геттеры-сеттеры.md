### способ № 1. Типовой

```python
def get_coords(self):
     return self.__x, self.__y

def set_coords(self, x, y):
     self.__x = x
     self.__y = y
```

### способ № 2. Через функцию property

```python
def get_coords(self):
  return self.__x, self.__y

def set_coords(self, x, y):
    self.__x = x
    self.__y = y


coords = property(get_coords, set_coords)

A = Point(1, 2)

print(A.coords)

```

### способ № 3 Через декоратор @property

```python
@property
 def x(self):
    return self.__x
@x.setter
def x(self, x):
       self.__x = x

print(A.x)

```

##  теперь понятно как передать другой объект класса! И поработать с его свойствами

```python
from math import hypot


class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def distance(self, other):
        return hypot(self.x - other.x, self.y - other.y)


p1 = Point(2, 7)
p2 = Point(12, 7)


dist = p1.distance(p2) # так ок

dist = Point.distance(p1,p2) # так не очень
```
