

```python
def h1(func):
    def wrapper():
        return '<h1>\n' + func() + '\n</h1>'
    return wrapper

def div(func):
    def wrapper():
        return '\t<p> ' + func() + ' </p>'
    return wrapper

@h1
@div
def hello():
    return 'Привет'


print (hello())
```


Результат
```
<h1>
	<p> Привет </p>
</h1>
```

Передача параметра в декоратор
```python
def h1(func):
    def wrapper(title):
        return '<h1>\n' + func(title) + '\n</h1>'
    return wrapper

@h1
@div
def hello(title):
    return title

print (hello('Привет'))

```
